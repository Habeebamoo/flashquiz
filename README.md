# 🎉 FlashQuiz  

**FlashQuiz** is a simple and interactive quiz web app powered by [Open Trivia Database](https://opentdb.com/). It provides a fun way to test your knowledge with various trivia questions while tracking your progress.  

## 🚀 Features  
- ✅ **Progress Tracking** – Monitor your quiz performance.  
- ✅ **Feedback System** – Get instant feedback on your answers.  
- ✅ **Stopwatch Time Limit** – Challenge yourself with a time-based quiz.  
- ✅ **Randomized Questions** – Never get the same quiz twice!  

## 🛠️ Built With  
- **JavaScript** (No backend, fully client-side)  
- **HTML & CSS**  
- **Open Trivia API** for fetching questions
